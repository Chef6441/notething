# Readme

## Features 
- F5 current time (Sydney)
- F6 Calendar
- Find Dialog (Ctrl + F)
- Find and Replace Dialog (Ctrl + H)
- Smart Home Key and Selection
N Pressing Shift + Home

	N Second time - Selects until the beginning of the line
